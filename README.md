# Features
* logging
    * colors
    * visual display of roll on "progress bar"
* Classes:
    * Plane - uses Correction Class
    * Environment - creates new Turbulence Events on plane
    * Event - abstract
        * Turbulence
        * Correction
* python generator used for random numbers
